## 🚀 Features

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/CharlesNeres/mini-commerce-cart.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
